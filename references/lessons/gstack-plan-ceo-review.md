challenge premises, expand scope when it creates a better product. Four modes:
SCOPE EXPANSION (dream big), SELECTIVE EXPANSION (hold scope + cherry-pick

## When Do You Use It?

- Use when asked to "think bigger", "expand scope", "strategy review", "rethink this",
- Proactively suggest when the user is questioning scope or ambition of a plan,

## What Do You Say to Claude?

Type `/plan-ceo-review` in Claude Code. That's it. The skill takes over from there.

**Trigger phrases that also work:**

## What Will Claude Ask You?

Claude will walk you through a structured conversation. Key questions include:

- *"what would make this 10x better for 2x the effort?"*
- *"what would make us fail?"*
- *"what should the user see first, second, third?"*
- *"what did the reviewer find?"*
- *"What happens when X is nil?"*
- *"When the API returns 429?"*
